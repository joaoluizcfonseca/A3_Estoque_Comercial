# A3_ESTOQUE_COMERCIAL
Repositórios do Trabalho A3 Algoritmos

Sistema de Controle de Estoque desenvolvido em Java como trabalho avaliativo (A3) da unidade curricular Algoritmos e Programação da Universidade do Sul de Santa Catarina – UNISUL. 

 

# Sobre o Projeto 

Qualquer empresa que trabalha com produtos precisa saber o que tem no estoque, quanto custam e quando está na hora de repor. Sem esse controle, é fácil perder dinheiro ou deixar clientes na mão. 

Pensando nisso, desenvolvemos um sistema de controle de estoque para uma empresa comercial fictícia. Pelo terminal, o usuário consegue cadastrar produtos, registrar entradas e saídas, reajustar preços e gerar relatórios — tudo organizado em menus simples e diretos. Os dados são armazenados em vetores durante a execução do programa. 

 

# Funcionalidades 

Cadastro de Produtos: incluir, alterar, consultar e excluir produtos do sistema 

Movimentação de Estoque: registrar entradas e saídas com atualização automática do saldo 

Reajuste de Preços:   aplicar percentual de reajuste em um produto específico ou em todos de uma vez 

Relatórios: Lista de Preços em ordem alfabética e balanço Físico-Financeiro com valor total do estoque

Organização geral / Main: implementação da classe `Main`, exibição do menu inicial e controle do fluxo de navegação entre os módulos do programa.
